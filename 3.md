# 第三題

## 題目
房間裡有三個袋子,一個只裝鉛筆,一個只裝原子筆,第三個有鉛筆也有原子筆,袋子是不透明的,單從
袋子的外表上看不出任何差異,你不知道哪一個袋子裝了什麼。除了袋子上各貼了一個標示("鉛筆"、"原子
筆"、"混和"),而且標示都是錯的(ex. 標有鉛筆的袋子一定不是只裝鉛筆)。

## solution
※ 以下以「」代表標示
拿「混合」的袋子。
因為所有標示都是錯的，所以「混合」的袋子只有兩種情況：

### 「混合」 = 鉛筆
* 「鉛筆」 = 原子筆
* 「原子筆」 = 混合

### 「混合」 = 原子筆
* 「原子筆」 = 鉛筆
* 「鉛筆」 = 混合
